# Week 6 Side Quest — Papers, Please Shooter

## Setup and Interaction Instructions

To run the sketch locally, open `index.html` in Google Chrome using Live Server.

**Controls:**

- Move: WASD
- Shoot: Spacebar
- B: Skip to boss fight (testing only)
- Restart: R (after win or game over)

Explore the world, survive enemy waves, and defeat the boss. The game has been redesigned using a Papers, Please theme with custom visuals, sounds, and obstacle data.

**Opening the Chrome Console**

- **Windows:** Press `F12` or `Ctrl + Shift + J`, then click the **Console** tab
- **Mac:** Press `Cmd + Option + J`

The console will show any errors in your sketch.
